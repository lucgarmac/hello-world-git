# Formatos de énfasis
Formato *itálica* forma 1.\
Formato _itálica_ forma 2.\
Formato **bold o strong** forma 1.\
Formato __bold o strong__ forma 2.\
Formato ~~tachado~~.\
Aquí podemos combinar los formatos *itálica*, **bold** y ~~tachado~~.
